Soal: Compare the Triplets
Kamu diberikan dua array yang masing-masing terdiri dari tiga elemen. Tugasmu adalah membandingkan elemen-elemen tersebut berdasarkan aturan berikut:

Jika elemen pada array pertama lebih besar dari elemen pada array kedua, orang pertama mendapat 1 poin.
Jika elemen pada array kedua lebih besar dari elemen pada array pertama, orang kedua mendapat 1 poin.
Jika kedua elemen sama, tidak ada yang mendapat poin.
Akhirnya, kamu harus mengembalikan dua nilai:

Skor total orang pertama.
Skor total orang kedua.
